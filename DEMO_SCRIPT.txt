INSTALLATION GUIDE (6:30 - 7:00)
--------------------------------
"If you want to run this application yourself:
1. Clone the repository
2. Create a virtual environment
3. Install Django 4.2.23 with 'pip install django'
4. Run migrations with 'python manage.py migrate'
5. Start the server with 'python manage.py runserver'
6. Access at localhost:8000

The application uses Bootstrap 4.6.2 for styling, jQuery for Bootstrap components, and SQLite3 for the database. Full setup instructions are in the README file."
